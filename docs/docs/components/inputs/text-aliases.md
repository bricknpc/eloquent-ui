# Text aliases

The text input component supports every input type available. To help you write more semantic code, we've added some 
aliases that set the type attribute for you. In all other ways they act the exact same as the text input component.

## List of aliases

### Color

```html
<x-eloquent-ui::input.color name="name" />
```

### Date

```html
<x-eloquent-ui::input.date name="name" />
```

### Datetime local

```html
<x-eloquent-ui::input.datetime-local name="name" />
```

### Email

```html
<x-eloquent-ui::input.email name="name" />
```

<x-eloquent-ui::input.month name="name" />
